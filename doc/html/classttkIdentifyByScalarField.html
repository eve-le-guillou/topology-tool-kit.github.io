Detailed Description

TTK VTK-filter that computes a new scalar array based on a sorting of the input array.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Florian Wetzels f_wet.nosp@m.zels.nosp@m.13@cs.nosp@m..uni.nosp@m.-kl.d.nosp@m.e
Date
June 2020

VTK code for computing a new scalar array from a specifiable point or cell data array. For each cell or a point the new (integer) value is its index in a sorting of the input array. The name of the new array is "PointScalarFieldName" or "CellScalarFieldName".

Parameters
InputInput scalar field (vtkDataSet)
OutputOutput scalar field (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

The input data array needs to be specified via the standard VTK call vtkAlgorithm::SetInputArrayToProcess() with the following parameters:

Parameters
idx0 (FIXED: the first array the algorithm requires)
port0 (FIXED: first port)
connection0 (FIXED: first connection)
fieldAssociation0 OR 1 (DYNAMIC: point or cell data)
arrayName(DYNAMIC: string identifier of the input array)
