Detailed Description

TTK VTK-filter to generate order fields.

Author
Pierre Guillou pierr.nosp@m.e.gu.nosp@m.illou.nosp@m.@lip.nosp@m.6.fr
Date
September 2020

This VTK filter generates order arrays from a selection of scalar field arrays.

Parameters
InputvtkDataSet.
OutputvtkDataSet.

See the related ParaView example state files for usage examples within a VTK pipeline.
