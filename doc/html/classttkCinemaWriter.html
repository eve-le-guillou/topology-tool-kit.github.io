Detailed Description

TTK VTK-filter that writes input to disk.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.09.2018

This filter stores the input as a VTK dataset to disk and updates the data.csv file of a Cinema Spec D database.

Parameters
InputvtkDataSet to be stored (vtkDataSet)

Online examples:

  • Cinema IO example
