Detailed Description

Composites multiple vtkImageData objects base on depth values.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.11.2020
Parameters
InputvtkImageData.
OutputvtkImageData.

Composites multiple vtkImageData objects passed to the first input port base on depth point data arrays specified via SetInputArrayToProcess(0,0,0,0,"DepthArrayName").

Related Publication: "Cinema Database Specification - Dietrich Release v1.2". D. Rogers, J. Woodring, J. Ahrens, J. Patchett, and J. Lukasczyk. Technical Report LA-UR-17-25072, Los Alamos National Laboratory, 2018.
