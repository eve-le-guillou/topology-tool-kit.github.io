Detailed Description

TTK VTK-filter that apply dimension reduction algorithms on input.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Date
September 2018
Parameters
InputInput scalar field (vtkTable)
OutputOutput scalar field (vtkTable)

TTK VTK-filter that takes a matrix (vtkTable) as input and apply a dimension reduction algorithm from scikit-learn.

See also
ttk::Triangulation
ttkDimensionReduction.cpp for a usage example.

Online examples:

  • 1-Manifold Learning example
  • Clustering Kelvin Helmholtz Instabilities example
  • Karhunen-Love Digits 64-Dimensions example
  • Merge Tree Clustering example
  • Merge Tree Principal Geodesic Analysis example
  • Persistent Generators Household Analysis example
  • Persistent Generators Periodic Picture example
  • TopoMap Teaser example
