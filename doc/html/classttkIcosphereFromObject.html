Detailed Description

Author
Jonas Lukasczyk (jl@jl.nosp@m.uk.d.nosp@m.e)
Date
01.09.2019

This filter creates an IcoSphere that encapsulates an input data object. The icosphere can optionally be scaled.

See also
ttk::IcoSphere
ttk::ttkAlgorithm
