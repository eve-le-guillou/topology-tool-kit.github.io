Detailed Description

The MergeTreeTemporalReductionDecoding class provides methods to compute the reconstruction of a reduced sequence of merge trees.

Author
Mathieu Pont (mathi.nosp@m.eu.p.nosp@m.ont@l.nosp@m.ip6..nosp@m.fr)
Date
2021.

This module defines the MergeTreeTemporalReductionDecoding class that computes the reconstruction of a reduced sequence of merge trees.

Related publication
"Wasserstein Distances, Geodesics and Barycenters of Merge Trees"
Mathieu Pont, Jules Vidal, Julie Delon, Julien Tierny.
Proc. of IEEE VIS 2021.
IEEE Transactions on Visualization and Computer Graphics, 2021
