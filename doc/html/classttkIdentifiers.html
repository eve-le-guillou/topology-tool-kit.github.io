Detailed Description

TTK VTK-filter that triggers the computation of global identifiers.

Author
Eve Le Guillou eve.l.nosp@m.e-gu.nosp@m.illou.nosp@m.@lip.nosp@m.6.fr
Date
October 2022.

This filter is useful when TTK is compiled with MPI to compute the MPI preconditioning of the Identifiers filter. When TTK is not compiled with MPI, the filter will compute the global identifiers without triggering the MPI preconditioning.

Parameters
InputInput data-set (vtkDataSet)
OutputOutput data-set with MPI preconditioning (vtkDataSet)

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

See the related ParaView example state files for usage examples within a VTK pipeline.
