Detailed Description

The ProjectionFromTable class provides methods that projects on a surface points in a vtkTable.

Author
Mathieu Pont mathi.nosp@m.eu.p.nosp@m.ont@l.nosp@m.ip6..nosp@m.fr
Date
2022.

This module defines the ProjectionFromTable class that projects on a surface points in a vtkTable.
