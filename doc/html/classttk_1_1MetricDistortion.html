Detailed Description

The MetricDistortion class provides methods to compute distance, area and curvature information about a surface and an optionnal distance matrix (giving the distance between the points of the surface in a metric space).

Author
Mathieu Pont mathi.nosp@m.eu.p.nosp@m.ont@l.nosp@m.ip6..nosp@m.fr
Date
2022.

This module defines the MetricDistortion class that computes distance, area and curvature information about a surface and an optionnal distance matrix (giving the distance between the points of the surface in a metric space).
